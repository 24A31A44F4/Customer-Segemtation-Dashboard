# 📊 Customer Segmentation Dashboard

## 📖 Project Overview

The **Customer Segmentation Dashboard** is an interactive Business Intelligence solution developed using **Microsoft Power BI**. This project analyzes customer demographics, annual income, spending behavior, and purchasing patterns using the **Mall Customer Segmentation Dataset**.

The dashboard provides meaningful insights into customer groups, helping businesses understand customer behavior and make data-driven marketing decisions through interactive visualizations.

---

## 🎯 Project Objectives

- Analyze customer demographics and purchasing behavior.
- Identify different customer segments based on income and spending score.
- Compare customer distribution by gender.
- Evaluate customer spending patterns across different age groups.
- Provide interactive filtering for detailed analysis.
- Support strategic business and marketing decisions.

---

## 📈 Key Performance Indicators (KPIs)

- 👥 Total Customers
- 🎂 Average Age
- 💰 Average Annual Income
- ⭐ Average Spending Score

---

## 📊 Dashboard Visualizations

- Customer Distribution by Gender (Donut Chart)
- Age vs Spending Score Analysis
- Average Income by Gender
